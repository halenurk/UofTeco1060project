# UofTeco1060project
Replication code files for Eco1060 project

WorknShirk.py generates the final simulation updated with feedback from presentations. 
It uses the final strategies for students after convergence for graphics.

AbsoluteGrading.py, CatalogSim.py, NestedSimRatioChanges.py, RelativeGrading.py, ProfsNestedSim.py, and RobustnessCheck.py were all parts of the code for that specific purpose. These got merged into a Master Simulation code later.
